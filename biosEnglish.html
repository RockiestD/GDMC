Accessing BIOS/UEFI

When turning on the computer, the system should go through the POST screen and then present a BIOS/UEFI screen. To access the BIOS/UEFI settings, press the indicated key (usually DEL or F2) right after the computer starts.


Initial settings


Checking the components


Processor (CPU): Check if it is correctly recognized.
RAM memory: Confirm that the installed quantity is correct.
Storage Devices (HD/SSD): Check if they are being recognized.
Video Card: Confirm that the graphics card is installed correctly (if any).


Configure the boot order


Adjust the boot order to ensure the computer boots from the installation USB stick/DVD or HD/SSD.


Enable XMP


XMP (for high-performance memory): Enable the XMP profile if applicable.
