Installing the operating system


If you haven't installed the operating system yet, now is the time:

Insert the installation device (pendrive or DVD with the operating system, such as Windows or Linux).


Start from Device

Turn on the computer and press the shortcut key for the boot menu (F12, ESC, or F11). If If necessary, adjust the boot order in BIOS/UEFI so that the pendrive/DVD is first.


Follow the installer instructions

The installer will load. Select the language, accept the license terms, and choose the partition where the system will be installed.
During installation, configure the username and password, and connect to the network.


Finish

After installation, your computer will restart. Remove the USB stick/DVD and follow the instructions to complete the settings. Check for system updates.
