Instalando o sistema operacional


Se você ainda não instalou o sistema operacional, esta é a hora:

Insera o dispositivo de instalação (pendrive ou DVD com o sistema operacional, como o Windows ou Linux).


Iniciar a Partir do Dispositivo

Ligue o computador e pressione a tecla de atalho para o menu de inicialização (F12, ESC, ou F11). Se necessário, ajuste a ordem de inicialização no BIOS/UEFI para o pendrive/DVD ser o primeiro.


Siga as intruções do instalador

O instalador será carregado. Selecione o idioma, aceite os termos de licença, e escolha a partição onde o sistema será instalado.
Durante a instalação, configure o usuário e a senha, e conecte-se à rede.


Finalizar

Após a instalação, o computador reiniciará. Remova o pendrive/DVD e siga as instruções para completar a configuração. Verifique se há atualizações do sistema.
