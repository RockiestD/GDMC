Installing the drivers

After installing the operating system, you will need to install essential drivers to ensure that your computer works properly:


Motherboard drivers

Install the chipset, audio, network and USB drivers, which usually come on the motherboard CD/DVD or can be downloaded from the manufacturer's website.


Video Card Drivers

Video card drivers (if you have one): If you are using a dedicated graphics card, download and install the drivers latest directly from the video card manufacturer's website (NVIDIA or AMD).


Operating system updates

Operating system updates: Run Windows Update or your operating system update to ensure that all security and driver updates are installed.
