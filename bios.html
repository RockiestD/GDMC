Acessando a BIOS/UEFI

Ao ligar o computador, o sistema deve passar pela tela POST e, em seguida, apresentar uma tela do BIOS/UEFI. Para acessar as configurações do BIOS/UEFI, pressione a tecla indicada (geralmente DEL ou F2) logo após o computador iniciar.


Configurações iniciais


Verificando os componentes


Processador (CPU): Verifique se está corretamente reconhecido.
Memória RAM: Confirme se a quantidade instalada está correta.
Dispositivos de Armazenamento (HD/SSD): Verifique se estão sendo reconhecidos.
Placa de Vídeo: Confirme se a placa gráfica está instalada corretamente (se houver).


Configure a ordem de boot


Ajuste a ordem de inicialização para garantir que o computador inicialize a partir do pendrive/DVD de instalação ou do HD/SSD.


Ative o XMP


XMP (para memória de alto desempenho): Ative o perfil XMP, se aplicável.
